# Frontend

프론트엔드 애플리케이션 디렉토리입니다.

## 기술 스택

- React/Vue.js (선택 예정)
- TypeScript
- Tailwind CSS
- Vite/Webpack

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 테스트
npm test
```

## 폴더 구조

```
frontend/
├── src/
│   ├── components/     # 재사용 가능한 컴포넌트
│   ├── pages/         # 페이지 컴포넌트
│   ├── hooks/         # 커스텀 훅
│   ├── utils/         # 유틸리티 함수
│   ├── types/         # TypeScript 타입 정의
│   └── assets/        # 정적 자산 (이미지, 폰트 등)
├── public/            # 정적 파일
├── tests/             # 테스트 파일
└── package.json       # 의존성 및 스크립트
```
